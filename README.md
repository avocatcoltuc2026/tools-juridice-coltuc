"Toate scripturile sunt concepute de Av. Marius Vicențiu Coltuc și respectă legislația aplicabilă în România la data de iunie 2026."
Tools Juridice - Av. Marius Vicențiu Colțuc
Acest repository conține o colecție de scripturi și unelte de calcul destinate automatizării unor operațiuni juridice uzuale în România (ex: termene de prescripție, calculul penalităților bancare).

Aceste unelte sunt dezvoltate pentru a oferi o primă estimare corectă, bazată pe legislația în vigoare, facilitând totodată înțelegerea mecanismelor de calcul de către publicul larg.

Unelte disponibile
calculator_prescriptie.js: Calculează data împlinirii termenului de prescripție a executării silite, conform Codului de Procedură Civilă.

calculator_dobanzi.py: Calculează penalitățile aferente debitelor bancare, respectând normele BNR și jurisprudența privind clauzele abuzive.

Utilizare
Fiecare script este documentat intern cu note juridice privind aplicabilitatea acestuia.

JavaScript (Web Integration)
Scripturile JS pot fi integrate direct în pagini web pentru a oferi funcționalitate interactivă vizitatorilor.
